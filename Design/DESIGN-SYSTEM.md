# BinO-Vault Design System

## Color Palette

### Primary Colors
- **Background:** #1A1A1A (Dark charcoal)
- **Primary Action:** #10B981 (Mint green)
- **Text Primary:** #FFFFFF (White)
- **Text Secondary:** #9CA3AF (Gray)

### Status Colors
- **Calm (Safe):** #10B981 (Green)
- **Alert (Warning):** #F59E0B (Amber/Yellow)
- **Critical (Danger):** #EF4444 (Red)

### UI Elements
- **Field Background:** #2D2D2D (Dark gray)
- **Border:** #374151 (Medium gray)

## Typography

- **Font Family:** Poppins
- **Weights Used:**
  - Regular (400) - Body text, placeholders
  - Medium (500) - Labels, secondary buttons
  - SemiBold (600) - Button text, card titles
  - Bold (700) - Headings, app name

- **Sizes:**
  - 36px - App name
  - 20px - Screen titles
  - 16px - Body text, button text, input text
  - 14px - Labels, secondary actions
  - 12px - Status badges, small labels

## Spacing

- **Base unit:** 4px
- **Common spacing:**
  - 8px - Tight spacing (badge to text)
  - 12px - Label to input
  - 16px - Padding inside fields/cards
  - 20-24px - Between form elements
  - 40-50px - Between major sections
  - 80-100px - Top spacing on screens

## Components

### Password Card
- Width: 370px

- Background: #2D2D2D
- Padding: 16px
- Status badge: 12px font, 8px padding

### Buttons
- Primary (green): 370px × 52px, #10B981, radius 12px
- Secondary (outline): 370px × 52px, 2px border, radius 12px

### Input Fields
- Width: 370px
- Height: 52px
- Background: #2D2D2D
- Corner radius: 12px
- Padding: 16px

## Logo

- Shield: 80px × 90px, #10B981, radius 16px
- Lock circle: 30px diameter, #1A1A1A
- Lock body: 28px × 20px, #1A1A1A
- Screws: 10px diameter, #FFFFFF (3 total in triangle formation)
